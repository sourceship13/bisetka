# Bisetka Splash Screen Setup

## Configuration

This app uses **`react-native-bootsplash`** with a custom full-screen background image.

### Assets

- **Logo (centered)**: `assets/bootsplash/logo.png` (88×191 @ 1x)
  - Variants: `@1.5x`, `@2x`, `@3x`, `@4x`
  
- **Full-screen background**: `assets/bootsplash/splash_fullscreen.png` (1440×2560)
  - Background color: `#0f0c29` (dark purple-blue)

### How It Works

1. **Android**: Uses `bootsplash_bg.png` (full-screen) + `bootsplash_logo.png` (centered)
2. **iOS**: Uses `BootSplashFullImage` (full-screen) + `BootSplashLogo-624c48` (centered) in the storyboard

## ⚠️ Important: Don't Break the Splash Screen

The `react-native-bootsplash generate` CLI **only creates a centered logo on a solid color** — it does NOT generate full-screen background images. Running it directly will **overwrite the iOS storyboard** and remove the full-screen image.

### Safe Way to Regenerate

**Always use the npm script:**

```bash
npm run bootsplash:generate
```

This runs:
1. `npx react-native-bootsplash generate` (standard generation)
2. `scripts/restore-bootsplash-fullscreen.js` (restores the full-screen image)

### Manual Restoration (if you accidentally broke it)

If you forgot and ran `npx react-native-bootsplash generate` directly:

```bash
node scripts/restore-bootsplash-fullscreen.js
```

## Updating the Splash Screen

### To change the background image:

1. Replace `assets/bootsplash/splash_fullscreen.png` with your new image (recommend 1440×2560)
2. Run `npm run bootsplash:generate`

### To change the centered logo:

1. Replace `assets/bootsplash/logo.png` (and variants)
2. Run `npm run bootsplash:generate`

### To change the background color:

1. Update the `--background` flag in [`package.json`](package.json) script
2. Run `npm run bootsplash:generate`

## Files Modified vs Generated

**Generated by bootsplash CLI (safe to regenerate):**
- `android/app/src/main/res/drawable-*/bootsplash_logo.png`
- `ios/bisetka/Images.xcassets/BootSplashLogo-*.imageset/`
- `android/app/src/main/res/values/colors.xml`

**Custom files (preserved by restoration script):**
- `assets/bootsplash/splash_fullscreen.png`
- `ios/bisetka/Images.xcassets/BootSplashFullImage.imageset/`
- iOS `BootSplash.storyboard` (full-screen image layer)
